Projeto interativo onde o usuário pode votar no seu filme favorito do universo Marvel e acompanhar o resultado em tempo real com barras de progresso animadas.

Sobre o Projeto:
Este projeto foi desenvolvido como parte dos meus estudos de HTML, CSS e JavaScript. O objetivo foi criar uma página interativa e estilizada do zero, aplicando conceitos de layout, animações, manipulação do DOM e lógica de programação.
